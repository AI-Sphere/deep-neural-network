#### AI圈是什么？

我们是一群通过数据竞赛认识的朋友，竞赛大佬们拿奖无数(不包括我)。比赛打累了，突然想做点情怀的事情。

以前我们一直孤独的在学习，直到我们相互遇见。所以我们想改变孤独，提供一个温暖的大家庭。

我们目前致力于提供一个非盈利平台一起集训学习编程、算法、数学、AI。这里没有小白歧视，大家一起互助共赢。

如果您有技术、运营、PS经历，可以公众号回复自己意愿和个人简介加入我们的核心团队。感谢阅读，感谢关注。
